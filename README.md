# VIZAR - Visa Services Website

A professional visa services website built with HTML, CSS, and JavaScript. This website helps users navigate visa applications for multiple countries including USA, Canada, Australia, UK, Japan, and Europe.

## Features

- Responsive design for desktop and mobile devices
- Country-specific visa information pages
- Interactive slider for country selection
- Mobile-friendly navigation with burger menu
- Comprehensive visa information for multiple destinations
- Contact forms and WhatsApp integration
